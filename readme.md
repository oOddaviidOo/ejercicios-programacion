# Ejercicios de Programación

La idea de este repositorio es ir subiendo ejercicios de programación para practicar mejorar nuestra lógica y agilidad a la hora de resolver problemas.

Cogeremos ejemplos de https://retosdeprogramacion.com e iremos completandolos semana a semana para ir comparando las soluciones de cada persona y así poder ver otra forma de afrontar el mismo problema.

> **Nota:** Se puede usar cualquier lenguaje de programación para resolver los ejercicios y cualquier editor de código.

## Instrucciones para hacer los ejercicios

* Lo primero será crear una rama con la siguiente nomenclatura:

    ejercicio-*nombre*-*numeroEjercicio*-*titulo_jercicio*

    > Ejemplo: ejercicio-david-1-fizz_buzz

* Una vez hecho el ejercicio, subimos la rama al repositorio.

* Tras esto, crearemos una pull request para solicitar mergear la rama del ejercicio en la rama *master* (podemos pedirle a Copilot o a David que revisen la pull request)
